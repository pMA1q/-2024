Japanese Hotel Room <Created by sbbututuya>

Description:
--------------------
With this package you can make a "Japanese hotel room".

This package contains:

- 118 meshs. (Polygon count : Min 2 - Max 4192)
- 197 prefabs.
- 99 materials.
- 200 textures. (1024x1024/2048x2048/4096x4096)
- 1 skybox cubemap material. (HDRI Haven/kiara_1_dawn_4k)
- 1 demo scene.
- 3 post processing stack v2 profiles.

Been used for AssetStore screenshots visualization:

- Post Processing Stack V2
- bathtub01_water(Object) : Water script and Water Pro Daytime material. (Standard Assets)
